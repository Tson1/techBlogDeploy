git 一个分支完全覆盖另一个分支

  1. git push origin develop:master -f
    就可以把本地的develop分支强制(-f)推送到远程master
  2. git checkout master // 切换到旧的分支
    git reset –hard develop // 将本地的旧分支 master 重置成 develop
    git push origin master –force // 再推送到远程仓库
